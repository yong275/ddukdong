#### 아동 발달 이론 기반 연령별 동화 생성 가이드라인

| 항목 | A구간 (만 4–6세) | B구간 (만 7–9세) | C구간 (만 10–12세) |
| --- | --- | --- | --- |
| **총 페이지 수** | 6–8컷 | 8–12컷 | 12–16컷 |
| **페이지당 문장 수** | 1–2문장 | 2–4문장 | 3–6문장 |
| **페이지당 최대 글자 수** | 40자 | 80자 | 150자 |
| **문장 구조** | 단문 위주, 단순 복문 허용 | 복문 허용, 연결어 사용 가능 | 복잡한 복문, 비유 표현 허용 |
| **어미·문체** | `~했어요 / ~이었어요 / ~했대요` 통일 | 자연스러운 구어체 | 자연스러운 구어체 |
| **의성어·의태어** | 페이지당 1회 이상 필수 | 적절히 사용 | 필요 시 사용 |
| **감정 표현** | 추상 감정어 금지, 신체 반응으로 표현 | 복합 감정 허용 | 다양한 감정과 심리 표현 가능 |
| **내면 서술** | 사용하지 않음 | 1~2회 포함 | 자유롭게 활용 |
| **반복 구조** | 3회 반복 패턴 필수 | 선택적 활용 | 필요 시 활용 |
| **새 어휘 처리** | 다음 문장에서 자연스럽게 의미 설명 | 문맥으로 이해 가능 | 별도 제한 없음 |
| **추상 가치·교훈** | 행동과 결과로만 표현 | 추상어 사용 시 행동 설명 동반 | 직접 언급 가능 (설교 금지) |
| **서브 캐릭터** | 단순 조력자 역할 | 보조 갈등 가능 | 감정선 병행 가능 |
| **결말** | 따뜻하고 안정적 마무리 | 문제 해결 중심 | 열린 결말·여운 있는 결말 허용 |
| **금지 표현** | 피동형 복문, 시제 역전, 추상 감정 명사 | 고등학교 수준 어휘, 한 문단 5문장 초과 | 설교조 표현 |

#### 전 연령 공통 금지 요소

| 구분 | 금지 내용 |
| --- | --- |
| 🚫 폭력 | 때리다, 죽이다, 피 등 신체적 해악 표현 |
| 🚫 편견 | 성별·외모 고정관념 강화 표현 |
| 🚫 차별 | 특정 집단 비하 및 차별 |
| 🚫 공포 | 지나친 공포 조성, "잡아간다" 등의 위협 |
| 🚫 보호자 | 부모·어른을 무능하게 묘사 |
| 🚫 안전 | 아이가 보호자 없이 위험한 상황에 처하는 장면 |

#### 연령 구간별 이미지 파라미터

| 파라미터 | A구간 (만 4–6세) | B구간 (만 7–9세) | C구간 (만 10–12세) |
| --- | --- | --- | --- |
| **사용 색상 수** | 3~4가지 (원색 위주) | 5~7가지 (파스텔 허용) | 제한 없음 (저채도 포함) |
| **배경 오브젝트 수** | 최대 3개 | 4~6개 | 제한 없음 |
| **배경 처리** | 단색 또는 단순 도형 | 반세밀 배경 | 세밀한 환경 묘사 |
| **캐릭터 크기** | 화면의 60% 이상 차지 | 화면의 40~60% | 자유 (환경 강조 가능) |
| **캐릭터 표정** | 과장되고 즉시 인식 가능 | 명확하고 섬세한 감정 표현 | 미묘하고 현실적인 감정 표현 |
| **원근감·깊이감** | 없음 (완전 평면) | 약하게 허용 | 적극 활용 |
| **그림자** | 없음 | 소프트 섀도 허용 | 자유 |
| **조명 효과** | 없음 | 없음 | 허용 |
