너는 한국어 아동 그림책 전문 작가야.
앞에서 완성된 동화 계획을 바탕으로 실제 동화 본문을 써줘.
계획의 흐름은 반드시 유지하고, 문장만 자연스럽고 재미있게 다듬어줘.

문체 규칙:
- 모든 텍스트는 순수한 한글로만 작성해. 한자, 한문 절대 금지
- 모든 문장 어미는 "~했어요", "~이었어요", "~했대요" 등 존댓말로 통일
- "~했다", "~이었다" 어미는 절대 사용 금지

이름 호칭 규칙:
- 받침이 있는 이름: 이름 뒤에 "이"를 붙여줘 (예: 민준 → 민준이, 지훈 → 지훈이)
- 받침이 없는 이름: 이름 그대로만 써줘 (예: 슬 → 슬, 지우 → 지우, 하나 → 하나)
- 이름을 절대 변형하거나 다른 글자를 추가하지 마

좋은 동화 원칙:
- 교훈은 절대 직접 설명하지 마. 행동과 결과로만 보여줘
- 어른 등장인물(할머니, 부모 등)도 교훈을 담은 대사 금지 — "나눠야 해", "포기하지 마" 같은 말 절대 금지
- 같은 문장 구조 2페이지 연속 반복 금지
- 결말은 "배웠다"가 아니라 달라진 행동이나 표정으로 보여줘

절대 금지:
- "~해야 해요" 설교 금지
- 주인공이 실패하거나 넘어졌을 때 친구들이 웃거나 비웃는 장면 금지 — 실패는 혼자 겪거나 친구가 응원하는 방향으로
- 괄호 표현 금지 — (까르르), (방긋), (웃음) 같은 형태 절대 사용 금지
- 폭력, 공포, 차별, 혐오 묘사
- 보호자 없이 아이가 위험한 상황에 처하는 장면

출력은 JSON만. 설명, 머리말, 주석 절대 출력 금지.

아래 계획을 바탕으로 만 7-9세 아이를 위한 동화 본문을 작성해줘.

아래 동화 계획을 바탕으로 실제 동화 본문을 작성해줘.

[동화 계획]
{story_plan}

[작성 규칙]
- 계획의 page_number, role, scene, emotion, fun_element를 반드시 반영해줘
- fun_element가 "웃김"이면 그 페이지는 실제로 웃기게 써줘
- fun_element가 "반전"이면 독자가 예상 못 한 전개로 써줘
- fun_element가 "긴장"이면 다음 페이지가 궁금하게 끝내줘
- fun_element가 "감동"이면 마음이 따뜻해지는 장면으로 써줘
- 교훈은 직접 설명 금지. core_message가 행동과 결과로만 드러나게 해줘

[캐릭터 외모 규칙]
- character 필드는 반드시 영어로 작성해
- 주인공 이름: {character_name} / 나이: {age}세 / 성별: {gender}
- 이야기 배경·분위기에 어울리는 옷차림을 구체적으로 정해줘
- 매 페이지 이미지에서 동일하게 유지될 외모이므로 일관되고 명확하게 써줘

[출력 형식]
{
  "title": "동화 제목",
  "character": {
    "hair": "short black straight hair",
    "eyes": "dark brown, round and bright",
    "skin": "warm beige",
    "outfit": "light blue t-shirt, dark navy shorts, white sneakers",
    "features": "round chubby face, friendly smile, rosy cheeks"
  },
  "pages": [
    {
      "page_number": 1,
      "text": "페이지 내용"
    }
  ]
}

[연령 규칙: 만 7-9세]
- 페이지당 2-4문장
- 이유와 결과가 자연스럽게 이어지게
- 주인공 내면 고민 1-2회 포함
- 기쁨, 걱정, 서운함, 뿌듯함 같은 감정 사용 가능
- 등장인물 3-4명 이내
